# 六足機械運動學模擬器 (Hexapod Kinematic Simulator)

這是一個基於 HTML5 Canvas 實作的六足機器人 2D 運動學模擬器。它展示了複雜的連桿機構、動態地面支撐邏輯以及物理平滑過渡。

## 核心功能

*   **動態支撐邏輯**：系統會自動根據足端位置計算最穩定的地面支撐線，模擬機器人在不平整地面上的傾角。
*   **雙視角模式 (Camera POV)**：
    *   **Robot-Centric**：機體保持水平，地面隨地形起伏與傾斜。
    *   **World-Centric**：地面保持絕對水平，機體隨運動產生真實的物理傾斜。
*   **物理平滑系統 (Spring-Damper)**：採用彈簧阻尼模型處理姿態轉換，使機器人的傾倒動作自然流暢，避免瞬間跳變。
*   **自動節能模式 (Auto-Sleep)**：針對行動裝置（iPad/手機）優化。當物理系統穩定且馬達停止時，會自動暫停渲染循環以節省電力。
*   **實時控制面板**：
    *   **Sim Speed**：控制馬達運轉角速度。
    *   **Stiffness**：調整機體回復平衡的剛性（反應速度）。
    *   **Manual Angle**：手動控制曲柄位置以觀察特定姿勢。

## 使用技術

*   **Vanilla JavaScript**：無外部依賴。
*   **HTML5 Canvas**：高效能 2D 渲染。
*   **CSS3**：響應式控制面板介面。

## 快速開始

直接在任何現代瀏覽器（Chrome, Safari, Firefox）中開啟 `六足機械模擬latest.html` 即可開始模擬。
