# 統一約束物理方案 (Unified Constraint-Based Physics)

為了更優雅地處理「單點支點旋轉」與「雙點支撐鎖定」，我們採用 **約束動力學 (Constraint Dynamics)** 來替代傳統的彈簧力（Penalty Method）。這能確保觸地點完全靜止，且自動處理多點間的相互影響。

## 1. 核心概念

我們不考慮「力」，而是考慮 **「衝量 (Impulse)」**。在每個子步驟中，我們詢問：
「需要施加多少衝量，才能讓觸地點的速度精準變為零？」

### 優點：
- **支點不跳動**：因為速度被精準歸零，木端不會因為彈簧回彈而跳起。
- **自動鎖定**：如果有兩點觸地，求解器會自動找到讓兩個點都靜止的解，從而導致機器人角度鎖定 ($\omega=0$)。
- **無縫過渡**：從自由落體、單點支撐到雙點鎖定，不需要 `if-else` 狀態機，完全由數學方程式自然導出。

---

## 2. 數學模型 (Simplified Solver)

對於每一個觸地點 $i$：
1. **目標速度更新**：
   我們希望 $v_{i,new} = v_{i,old} + \Delta v_i = 0$。
2. **衝量方向**：
   地面提供一個向上的衝量 $J_i$。
3. **對全身的影響**：
   - $\Delta v_{com} = J_i / M$
   - $\Delta \omega = (r_i \times J_i) / I$
4. **求解 $J_i$**：
   聯立方程，求出能補償 $v_{i,old}$ 垂直與水平分量的 $J$。

---

## 3. 算法流程 (The Algorithm)

1. **預測運動 (Predict)**：
   正常計算重力、馬達力矩產生的加速度，更新預測速度。
2. **識別約束 (Identify)**：
   檢測哪些點穿透了地面。
3. **衝量求解 (Solve)**：
   如果是雙點觸地，聯立求解 $(J_{wood}, J_{pvc})$，使得兩點速度皆歸零。
4. **應用與修正 (Apply)**：
   更新質心速度與角速度。如果計算出的 $J$ 是負值（代表需要拉住地面），則取消該約束（代表離地）。

---

## 4. 預期效果與你的觀察對照

- **PVC 擠壓反彈時**：
  系統會計算出一個衝量 $J_{pvc}$。如果此時木腿也在地上，系統會發現為了維持木腿不動，能量必須轉化為繞木腿的虛擬扭矩，這會使機器人自然地繞著木腿旋轉。
- **雙點平穩支撐時**：
  $J_{wood}$ 與 $J_{pvc}$ 會同時運算，抵銷掉所有的垂直運動與轉動，使機器人穩定的「鎖定」在地面上，直到馬達產生的力矩克服了其中一端的重力矩。

這套方案將提供目前為止最真實、最「重」的物理回饋感。
