# 六足機器人模擬器：數學原理綜述

本文檔彙整了「六足機械模擬」程式中所使用的核心數學邏輯，包含連桿運動學、動態地面支撐判斷、彈簧阻尼平滑系統以及多視角座標轉換。

---

## 1. 連桿幾何與運動學 (Linkage Kinematics)

機器人的單腿由機體固定點、曲柄銷以及多個連桿組成。其核心運算在於求解兩個圓的交點。

### 圓與圓的交點 (getIntersection)
當已知兩個圓的圓心 $C1, C2$ 與半徑 $r1, r2$ 時，交點 $P$ 的計算如下：
1.  計算圓心距離 $d = \sqrt{(C2.x-C1.x)^2 + (C2.y-C1.y)^2}$。
2.  利用餘弦定理求出從 $C1$ 到交點連線在 $d$ 軸上的投影長度 $a$：
    $a = (r1^2 - r2^2 + d^2) / (2d)$
3.  求出垂直於 $d$ 軸的高度 $h$：
    $h = \sqrt{r1^2 - a^2}$
4.  根據幾何關係求出可能的兩個交點 $P3a, P3b$。程式中選擇 $y$ 座標較大者（在畫布中通常代表特定結構方向）。

### 點的延伸 (extendPoint)
為了計算足端 (Foot) 位置，程式將「關節 A」到「關節 B」的向量延伸固定長度 $extLen$：
-   向量 $V = B - A$
-   單位向量 $u = V / |V|$
-   足端位置 $P = B + u \times extLen$

## 2. 動態地面支撐邏輯 (Ground Support Detection)

程式會從前 (Front)、中 (Middle)、後 (Rear) 三隻腳中找出最符合物理穩定性的支撐線。

### 支撐線判定準則
對於任意兩隻腳組成的直線，必須滿足以下條件才能視為地面：
1.  **涵蓋重心 (Stability)**：機體重心 ($x=0$) 必須位於兩支撐點的 $x$ 範圍之間。即 $min(x_a, x_b) \le 0 \le max(x_a, x_b)$。
2.  **外凸包性質 (Convex Hull)**：所有其餘的腳必須位於該直線的「上方」或「線上」。在程式坐標系中，這代表其 $y$ 值不小於直線在該點的預測值。

### 求解直線方程
已知兩點 $(x_a, y_a)$ 與 $(x_b, y_b)$：
-   斜率 $m = (y_b - y_a) / (x_b - x_a)$
-   截距 $c = y_a - m \times x_a$

## 3. 彈簧阻尼平滑系統 (Spring-Damper Smoothing)

當支撐腳切換時，目標地面線會發生瞬間跳變。為了使機器人傾倒動作流暢，程式採用了二階常微分方程（彈簧阻尼模型）來驅動地面參數 $m$ (斜率) 與 $c$ (截距)。

### 動態方程式
系統受力模型如下：
-   **回復力 (Spring Force)**: $F_s = k \times (target - current)$
-   **阻尼力 (Damping Force)**: $F_d = -b \times velocity$

其中系統參數與模擬速率 $simSpeed$ 關聯：
1.  **剛性 (Stiffness)**: $k = stiffness\_slider \times simSpeed$
2.  **臨界阻尼 (Critical Damping)**: $b = 2 \times damping\_ratio \times \sqrt{k}$
    （程式中 $damping\_ratio$ 固定為 0.75，提供接近臨界但略微欠阻尼的平滑感）

### 數值積分
每一幀更新：
-   加速度 $acc = k \times (error) - b \times velocity$
-   速度 $velocity = velocity + acc$
-   位置 $position = position + velocity$

## 4. 視角轉換矩陣 (POV Coordinate Mapping)

程式提供兩種視角切換，透過 `mapCoords` 函數實作 2D 仿射變換。

### 機體視角 (Robot-Centric)
機體固定於 $(0,0)$ 且不旋轉（視覺水平）。
-   直接將局部座標 $(x, y)$ 對應至畫布中心。
-   地面線根據計算出的 $m, c$ 進行傾斜繪製。

### 世界視角 (World-Centric)
地面固定於水平位置，機體隨地面傾斜。
1.  **旋轉角度**: $\theta = -arctan(m_{smoothed})$。
2.  **坐標旋轉**:
    $x' = x \times cos(\theta) - y \times sin(\theta)$
    $y' = x \times sin(\theta) + y \times cos(\theta)$
3.  **垂直位移**:
    為了使地面保持在固定高度，機體需根據 $c_{smoothed}$ 的偏移值進行垂直校正，並補償旋轉帶來的投影位移。

本模擬器結合了靜力學穩定分析與動力學平滑濾波，確保了在不同視角下皆能精確且流暢地呈現六足步行之機體姿態。
