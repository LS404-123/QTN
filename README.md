# 六足機械運動學模擬器 (Hexapod Kinematic Simulator)

這是一個基於 HTML5 Canvas 實作的六足機器人 2D 運動學模擬器。它展示了複雜的連桿機構、動態地面支撐邏輯以及物理平滑過渡。

## 核心功能

*   **動態支撐邏輯**：系統會自動根據足端位置計算最穩定的地面支撐線，模擬機器人在不平整地面上的傾角。
*   **雙視角模式 (Camera POV)**：
    *   **Robot-Centric**：機體保持水平，地面隨地形起伏與傾斜。
    *   **World-Centric**：地面保持絕對水平，機體隨運動產生真實的物理傾斜。
*   **物理平滑系統 (Spring-Damper)**：採用彈簧阻尼模型處理姿態轉換，使機器人的傾倒動作自然流暢，避免瞬間跳變。
*   **自動節能模式 (Auto-Sleep)**：針對行動裝置（iPad/手機）優化。當物理系統穩定且馬達停止時，會自動暫停渲染循環以節省電力。
*   **實時控制面板**：
    *   **Sim Speed**：控制馬達運轉角速度。
    *   **Gravity Scale**：影響機體回復平衡的重力係數（模擬物體的沉重感）。
    *   **Manual Angle**：手動控制曲柄位置以觀察特定姿勢。

## 重要變數說明

### 幾何與環境常數
*   `S`, `R`, `L_leg`, `L_blue`, `L_foot` (L133)：描述連桿機構長度與曲柄半徑。
*   `groundY` (L134)：環境中的標準水平地面 Y 座標。

### 狀態與物理變數
*   `theta` (L138)：當前曲柄旋轉角度（弧度）。
*   `simSpeed` (L141)：模擬運行速度。
*   `gravityScale` (L142)：重力加速度係數，影響機體倒向地面的速度。
*   `smoothedGround` (L149)：封裝了平滑後的地面斜率 (`m`) 與偏移量 (`c`)。
*   `angularVel` / `linearVelC` (L150-151)：機體傾斜的角速度與垂直平移速度。

### 物理穩定機制
*   **摩擦係數** (L344-345)：對 `angularVel` 與 `linearVelC` 施加 0.96 的衰減係數。
*   **碰撞偵測** (L349-352)：透過 `prevDiff` 與 `currDiff` 監測機體是否越過目標地面線。

## 座標系與物理方向

### 模擬座標系 (Simulation Coordinates)
本模擬器內部使用標準笛卡爾座標系（原點在機體中心）：
*   **X 軸**：正值 (+) 向**右**，負值 (-) 向**左**。
*   **Y 軸**：正值 (+) 向**上**，負值 (-) 向**下**。
    *   *註：經由 `mapCoords` 轉換後，畫布上的 Y 軸會被垂直翻轉，以符合直覺。*

### 數值正負號意義
*   **角度 `theta`**：遵循單位圓定義，遞增時為**逆時針**旋轉。
*   **斜率 `m`**：
    *   `m > 0`：地面向右上方傾斜（右側較高）。
    *   `m < 0`：地面向右下方傾斜（右側較低）。
*   **角速度 `angularVel`**：
    *   正值 (+)：正在增加斜率（嘗試抬高右側或壓低左側）。
    *   負值 (-)：正在減小斜率（嘗試壓低右側或抬高左側）。

## 使用技術

*   **Vanilla JavaScript**：無外部依賴。
*   **HTML5 Canvas**：高效能 2D 渲染。
*   **CSS3**：響應式控制面板介面。

## 快速開始

直接在任何現代瀏覽器（Chrome, Safari, Firefox）中開啟 `六足機械模擬latest.html` 即可開始模擬。
